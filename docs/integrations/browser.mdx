## Setup

Set your browser type:

```bash
export NTRP_BROWSER=chrome   # or: safari, arc
```

<Note>macOS only. ntrp reads the browser's SQLite history database directly.</Note>

## Configuration

| Variable | Default | Description |
|----------|---------|-------------|
| `NTRP_BROWSER` | — | Browser type: `chrome`, `safari`, or `arc` |
| `NTRP_BROWSER_DAYS` | `30` | Days of history to include |

## Available tools

| Tool | Description |
|------|-------------|
| `browser` | Search and list recent browser history |

## How it works

The browser source reads your browser's local SQLite database to find recently visited pages. This gives the agent context about what you've been researching or reading.

The agent can use this to:
- Find a page you visited recently but can't remember
- Get context about topics you've been researching
- Cross-reference browser activity with other sources
